#Technical Requirements

This projects requires:
- Java 1.8
- MySQL 5.6
- MQTT Broker (running)
- Maven

Database configurations can be made in
	/src/main/resources/hibernate.cfg.xml
	
	
#General Functions
- Register Screen



- Training Screen



- WoZ Controller



- Settings


#Compilation

Create an executable *.jar by executing:

	mvn clean compile assembly:single
